# 🥊 Boxing Arena System

## 🚀 Overview
IoT-based real-time boxing scoring and practice analytics system using ESP32 and Firebase.

## 🔥 Features
- Live red & blue scoring
- Round timer system
- Match category selection (U14, U17, U19)
- Practice mode with punch tracking
- Average punch gap calculation
- Firebase real-time updates

## 🛠 Tech Stack
- ESP32
- Firebase Realtime Database
- HTML, CSS, JavaScript

## 📊 How It Works
- FSR sensor detects punch force
- ESP32 sends data to Firebase
- Website reads data and updates live score
